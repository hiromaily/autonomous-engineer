# Product Overview

Autonomous Engineer is a spec-driven autonomous engineering system that orchestrates AI agents to execute the full software development lifecycle — from specification to production-ready code — with minimal human intervention.

## Core Capabilities

- **Spec-Driven Development (SDD)**: Structured workflow from `spec-init` → requirements → design → tasks → implementation
- **Workflow Orchestration**: State-driven engine that manages each development phase with clear transitions
- **AI Agent Execution**: Uses LLMs to generate specifications, implement features, review code, and refine designs
- **Persistent Memory**: Stores project rules, patterns, and past decisions to improve over time
- **Self-Healing Loops**: Analyzes failures and updates internal rules to prevent recurrence

## Target Use Cases

- Autonomous execution of a feature from specification to pull request
- Spec-driven development workflows with structured phase reviews
- AI-assisted development with persistent knowledge accumulation across sessions

## Value Proposition

Unlike stateless AI coding assistants, Autonomous Engineer acts as a **workflow orchestrator** — coordinating structured development phases, maintaining memory across sessions, and enabling autonomous execution with human review gates at each phase.

The system supports multiple SDD frameworks (cc-sdd, OpenSpec, SpecKit) and is designed to evolve toward multi-agent team architectures.
